# Travel App Frontend

Frontend built with Next.js 14 (App Router).

## Structure

```
src/
├── app/              # Next.js App Router pages
├── components/       # React components
├── modules/          # Business modules (feature-based)
└── shared/           # Shared utilities, types, constants
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
